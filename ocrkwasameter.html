OCR

This is a simple OCR application that can be used to recognize text from images.

To use the application, simply upload an image and click the "Recognize" button. The application will then recognize the text in the image and display it below the image.

You can also use the application to recognize text from a webcam. To do this, simply click the "Webcam" button and the application will start recognizing text from the webcam.

The application is currently in development, so there may be some bugs. Please let me know if you find any bugs or have any suggestions.

Thank you for using the application!
